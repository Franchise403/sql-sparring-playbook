SQL Sparring Playbook — structured SQL practice rounds from beginner to advanced, built using the **AdventureWorksLT2022 (SalesLT)** sample database.

This repository is designed as a portfolio-style study system, with each level containing:

- A **Question Bank** (prompt-based sparring rounds)
- An **Answer Key** (.sql solutions)
- Clear focus areas (Fundamentals → Joins → Advanced Querying)

Dataset used:  
AdventureWorksLT2022 (Microsoft sample database)  
https://learn.microsoft.com/en-us/sql/samples/adventureworks-install-configure

Levels Completed:
- Level 1 — Fundamentals (SELECT, WHERE, ORDER BY, LIKE, Aggregates)
- Level 2 — Intermediate SQL (IN/BETWEEN, GROUP BY, HAVING, Multi-column sorting)
- Level 3 — Joins & Relationships (INNER JOIN, LEFT JOIN, Multi-table joins)

Remaining advanced levels (CTEs, subqueries, window functions) coming next.
