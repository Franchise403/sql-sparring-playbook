# SQL Sparring Playbook

Structured SQL practice rounds from beginner to advanced, built using the  
**AdventureWorksLT2022 (SalesLT)** sample database.

This repository is designed as a portfolio-style study system, with each level containing:

- A **Question Bank** (prompt-based sparring rounds)
- An **Answer Key** (.sql solutions)
- Clear progression from fundamentals → joins → advanced querying → reporting SQL

---

## Dataset Used

**AdventureWorksLT2022 (Microsoft Sample Database)**  
https://learn.microsoft.com/en-us/sql/samples/adventureworks-install-configure

---

## Levels Completed

- **Level 1 — Fundamentals**  
  SELECT, WHERE, ORDER BY, LIKE, Aggregates

- **Level 2 — Intermediate SQL**  
  IN, BETWEEN, GROUP BY, HAVING, Multi-column sorting

- **Level 3 — Joins & Relationships**  
  INNER JOIN, LEFT JOIN, Multi-table joins

- **Level 4 — Advanced Querying**  
  CASE, CTEs, Window Functions, Ranking, Running Totals

- **Level 5 — Real-World BI SQL (Reporting & KPIs)**  
  Revenue summaries, Customer lifetime value, Category rollups, MoM trends, Executive KPI reporting
